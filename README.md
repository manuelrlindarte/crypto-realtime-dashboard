# Cripto en vivo — Dashboard de precios en tiempo real

Este proyecto es un dashboard interactivo desarrollado con código que muestra la fluctuación de precios de criptomonedas en **tiempo real**, utilizando feeds de mercado vía WebSockets.

El objetivo del dashboard es comunicar, a un **público general**, cómo cambian los precios de los activos digitales segundo a segundo, destacando la naturaleza dinámica y efímera de los datos financieros en vivo.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- D3.js
- WebSockets
- APIs de mercado (CoinCap y Binance)

---

##  ¿Qué muestra el dashboard?

- Estado de conexión en tiempo real
- Hora local y última actualización
- Contador de mensajes recibidos
- Precio actual del activo seleccionado
- Gráfica de línea que muestra la **fluctuación del precio** en una ventana temporal reciente (~60 puntos)
- Comparación visual entre distintos activos

> El eje temporal representa **instantes consecutivos en segundos**, no horas históricas, lo que permite observar el “ritmo” del mercado en tiempo real.
